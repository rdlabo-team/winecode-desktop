# winecode desktop

スマホで使える業務用ワイン管理アプリ winecodeのデスクトップアプリをダウンロードいただけます。
現在ベータ版で運用しております。ほとんどの機能はご利用いただけますが、特定の環境で一部機能が動かない可能性があります。ご了承ください。

[ダウンロードはこちらから](https://github.com/rdlabo-team/winecode-desktop/releases/latest)
